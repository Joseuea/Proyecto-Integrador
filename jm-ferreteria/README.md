# JM Ferretería 🔧

Sitio web de **JM Ferretería**, un negocio familiar ubicado en Carapungo (Panamericana Norte, Quito) dedicado a la venta de materiales de construcción y a servicios de fletes, mudanzas y retiro de escombros. Su lema es *"Construyendo tus sueños"*.

## Descripción

La página presenta la información del negocio y permite a los clientes conocer los productos y solicitar cotizaciones en línea. Incluye las siguientes secciones:

- **Inicio:** presentación del negocio.
- **Quiénes Somos:** historia y descripción de la ferretería, con su propietario y administrador.
- **Productos y Servicios:** catálogo de materiales (hierro, cemento, bloques, tuberías, arena, ripio, etc.) y servicios de transporte.
- **Cotizaciones:** formulario donde el cliente registra su solicitud; la lista se actualiza en pantalla sin recargar la página.
- **Contacto:** teléfonos, correo, dirección, redes sociales y un video.

## Tecnologías utilizadas

- **HTML5** con etiquetas semánticas
- **CSS3** con diseño responsivo (media queries)
- **Bootstrap 5** para el menú, tarjetas, formularios y sistema de grillas
- **JavaScript** para las validaciones del formulario y el manejo dinámico de las solicitudes (registrar, mostrar, contar y eliminar)

## Cómo ver la página

Abrir el archivo `index.html` en cualquier navegador web, o usar la extensión Live Server de Visual Studio Code.
